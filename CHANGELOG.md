# Changelog

Todas las modificaciones importantes del programa se documentan aquí.

## [1.1.0] - 2026-03-13

### Added
- Visual badge displaying the application version in the header.

- Support for VERSION files for runtime version detection.

### Improved
- The included terminal now displays the start and end of terminal/server scans.

- Improvements to the UI header design.

### Fixed
- Clicking on a port option now auto-completes.
